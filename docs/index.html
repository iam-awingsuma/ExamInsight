Project Overview

ExamInsight is a data analytics platform designed to analyze and visualize student assessment performance across cohorts, year groups, and individual students.

The system integrates standardized assessments such as NGRT and provides dashboards for educators to track progress, identify trends, and support data-driven instruction.

⚙️ Technology Stack

  • Python Flask
  • SQLAlchemy
  • Bootstrap UI
  • Chart.js / Plotly
  • REST APIs

📈 Features

  • Cohort performance analytics
  • Year group insights
  • Student progress tracking
  • External assessment analysis
  • Interactive dashboards

🧠 Architecture

  • Flask backend API
  • Jinja template rendering
  • JavaScript chart modules
  • REST analytics endpoints
